
# Yijun's Personal Portfolio

A modern, responsive personal portfolio website built to showcase my projects, skills, and professional journey as a Data Scientist and AI Engineer.

**Live Demo:** [yijunzhou99.github.io/portfolio/](https://yijunzhou99.github.io/portfolio/)
               or [https://yijun-portfolio.vercel.app/](https://yijun-portfolio.vercel.app/)

## 🚀 Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** CSS3
- **Deployment:** Vercel / GitHub Pages

## 🛠️ Features

- **Project Gallery:** A curated list of data science and AI engineering projects.
- **Technical Blog:** Integration for sharing insights on RAG, autonomous agents, and DSPy.
- **Responsive Design:** Optimized for viewing on desktops, tablets, and mobile devices.
- **Contact Integration:** Easy links to professional social profiles and email.
